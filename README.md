# Smart Fire Source Identification System

## Description
This project detects and classifies fire sources using multi-gas sensor fusion.

## Components Used
- ESP32
- MQ-2, MQ-7, MQ-135 sensors

## Working
The system reads gas values and classifies fire type such as wood, plastic, electrical, and gas leaks.

## Features
- Real-time detection
- Fire type classification
- Buzzer and LED alert
